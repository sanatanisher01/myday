About Templates

Newsletter templates allow you to create reusable designs for your email campaigns. You can:

  • Create multiple templates for different types of newsletters
  • Set a default template that will be pre-selected when creating a new campaign
  • Use HTML to customize the appearance of your newsletters
  • Include variables that will be replaced with actual content when sending
Available Variables

You can use these variables in your templates:

  • {{ "{{ name }}" }} - Recipient's name
  • {{ "{{ email }}" }} - Recipient's email address
  • {{ "{{ subject }}" }} - Email subject
  • {{ "{{ custom_message }}" }} - Custom message for this campaign
  • {{ "{{ unsubscribe_url }}" }} - Link to unsubscribe
